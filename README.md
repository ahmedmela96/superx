# SUPERX Shipping Management System

نموذج MVP لموقع شركة الشحن **SUPERX** يحتوي على:

- لوحة الإدارة: متابعة التجار، المندوبين، الشحنات، والمرتجعات.
- لوحة التاجر: إنشاء شحنة جديدة ومتابعة حالة الشحنات.
- واجهة العميل: تتبع الشحنة برقم التتبع.
- واجهة المندوب: تحديث الحالة (تسليم / فشل / تأجيل / مرتجع).

## التشغيل

افتح ملف `index.html` مباشرة في المتصفح.

## ملاحظات

- البيانات تجريبية ويتم حفظها محلياً في `localStorage`.
- المشروع Frontend فقط ويمكن ربطه لاحقاً بـ API و قاعدة بيانات.
